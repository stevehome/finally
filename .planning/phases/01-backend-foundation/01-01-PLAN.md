    1. `users_profile` — id TEXT PRIMARY KEY, cash_balance REAL NOT NULL DEFAULT 10000.0, created_at TEXT NOT NULL
    2. `watchlist` — id TEXT PRIMARY KEY, user_id TEXT NOT NULL DEFAULT 'default', ticker TEXT NOT NULL, added_at TEXT NOT NULL, UNIQUE(user_id, ticker)
    3. `positions` — id TEXT PRIMARY KEY, user_id TEXT NOT NULL DEFAULT 'default', ticker TEXT NOT NULL, quantity REAL NOT NULL, avg_cost REAL NOT NULL, updated_at TEXT NOT NULL, UNIQUE(user_id, ticker)
    4. `trades` — id TEXT PRIMARY KEY, user_id TEXT NOT NULL DEFAULT 'default', ticker TEXT NOT NULL, side TEXT NOT NULL, quantity REAL NOT NULL, price REAL NOT NULL, executed_at TEXT NOT NULL
    5. `portfolio_snapshots` — id TEXT PRIMARY KEY, user_id TEXT NOT NULL DEFAULT 'default', total_value REAL NOT NULL, recorded_at TEXT NOT NULL
    6. `chat_messages` — id TEXT PRIMARY KEY, user_id TEXT NOT NULL DEFAULT 'default', role TEXT NOT NULL, content TEXT NOT NULL, actions TEXT, created_at TEXT NOT NULL

    **init_db() function:**
    - `os.makedirs(os.path.dirname(os.path.abspath(DB_PATH)), exist_ok=True)`
    - Connect, execute _SCHEMA_SQL via executescript, call _seed(conn), conn.commit()

    **_seed(conn) function:**
    - `INSERT OR IGNORE INTO users_profile (id, cash_balance, created_at) VALUES (?, ?, ?)` with ("default", 10000.0, now_iso)
    - For each ticker in _DEFAULT_TICKERS: `INSERT OR IGNORE INTO watchlist (id, user_id, ticker, added_at) VALUES (?, ?, ?, ?)` with (uuid4_str, "default", ticker, now_iso)
    - Use `datetime.now(timezone.utc).isoformat()` for timestamps and `str(uuid.uuid4())` for IDs

    **get_watchlist_tickers(user_id="default") -> list[str]:**
    - Connect to DB_PATH, SELECT ticker FROM watchlist WHERE user_id = ? ORDER BY added_at, return list of ticker strings

    **get_connection() -> sqlite3.Connection:**
    - Connect to DB_PATH, set conn.row_factory = sqlite3.Row, return conn

    Per D-02: DB file at `db/finally.db` (relative, overridden by DB_PATH env var for Docker/tests).
    Per D-03: stdlib sqlite3 only, no ORM.
    Per D-04: CREATE TABLE IF NOT EXISTS + INSERT OR IGNORE for idempotency.
