  <files>backend/app/routers/portfolio.py</files>
  <behavior>
    - GET /api/portfolio returns {"cash_balance": float, "positions": [...], "total_value": float}
    - Each position object: {ticker, quantity, avg_cost, current_price, unrealized_pnl, value}
    - current_price falls back to avg_cost if PriceCache has no entry for that ticker
    - POST /api/portfolio/trade accepts {ticker: str, quantity: float, side: "buy"|"sell"}
    - Buy: validates cash >= quantity * current_price, else raises HTTPException(400)
    - Sell: validates existing position quantity >= requested quantity, else raises HTTPException(400)
    - Buy/Sell mutate cash, upsert position (weighted avg cost on buy), insert trade — all in one SQLite transaction
    - On buy: if position exists, new_avg = (old_qty * old_avg + qty * price) / (old_qty + qty)
    - On sell: if position.quantity reaches 0 after sell, DELETE the position row
    - After trade completes, call record_snapshot(price_cache) to insert a portfolio_snapshots row
    - POST returns {"status": "ok", "ticker": str, "side": str, "quantity": float, "price": float}
    - GET /api/portfolio/history returns {"snapshots": [{"total_value": float, "recorded_at": str}, ...]}
    - record_snapshot(price_cache) computes total_value = cash + sum(qty * price for each position) and inserts a row
  </behavior>
